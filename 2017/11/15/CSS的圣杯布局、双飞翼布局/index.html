CSS的圣杯布局、双飞翼布局

发表于 2017-11-15 | 分类于 CSS

圣杯布局和双飞翼布局解决的问题：两边顶宽，中间自适应的三栏布局，中间栏要放在文档流前面先渲染

两种布局在解决问题的前面部分是一致的，都是三栏全部浮动，左右两列加上负margin让其与中间div并排，并形成三栏布局

两者的不同在于

圣杯布局，为了中间div内容不被遮挡，

  1. 三者均设置左浮动；
  2. 设置middle的宽度为100%；
  3. 设置负边距，left负左边距设置100%，right负左边距为它本身的宽度
  4. 设置container左右的padding值给左右两个子面板留出空间
  5. 设置left和right为相对定位，position：relative，“left”的left值为负的”left”的宽度，“right”的right值为负的“right”宽度
    效果图

    存在的问题

    当中间部分的宽度小于两边的宽度时，布局就会乱掉

此时，双飞翼就驾着七彩祥云来了，


双飞翼布局是在middle的div里插入一个div,通过调整内部div的margin值，实现中间栏自适应，内容写到内部div中
